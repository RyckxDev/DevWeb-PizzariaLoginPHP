# 🍕 Pizzaria Online - Sistema de Login e Carrinho

Este é um projeto de desenvolvimento web focado na criação de uma interface funcional para uma pizzaria, integrando um sistema de autenticação de usuários e gerenciamento de carrinho de compras utilizando **PHP**.

## 🚀 Tecnologias Utilizadas

O projeto foi desenvolvido com as seguintes tecnologias:

* **PHP**: Lógica do lado do servidor e gerenciamento de sessões.
* **CSS3**: Estilização e responsividade (arquivos `login.css` e `response.css`).
* **HTML5**: Estruturação das páginas.

## 📂 Estrutura do Projeto

O repositório contém os seguintes arquivos principais:

* `login.php`: Página de autenticação de usuários.
* `dashboard.php`: Painel principal acessado após o login (exibição de produtos/pizzas).
* `carrinho.php`: Gerenciamento dos itens selecionados pelo usuário.
* `logout.php`: Encerramento seguro da sessão do usuário.
* `login.css` / `response.css`: Arquivos de estilo para garantir um visual atraente e adaptável a diferentes telas.

## 🛠️ Funcionalidades

- [x] **Sistema de Login**: Autenticação para acesso às áreas restritas.
- [x] **Catálogo de Produtos**: Visualização das opções disponíveis na pizzaria através do Dashboard.
- [x] **Carrinho de Compras**: Adição de itens para simulação de pedido.
- [x] **Sessões Seguras**: Controle de acesso por meio de sessões PHP.
- [x] **Design Responsivo**: Layout que se adapta a dispositivos móveis.
